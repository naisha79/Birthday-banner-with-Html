# Birthday Card Webpage 🎂

A simple birthday greeting webpage built with HTML and CSS.
The page displays a styled birthday card with a floating animation and a short message.

## Features

* Styled birthday card layout
* Soft pastel color theme
* Floating CSS animation
* Emoji decorations

## Languages

* HTML
* CSS

## How to Use

1. Download the project files
2. Open `index.html` in any web browser
